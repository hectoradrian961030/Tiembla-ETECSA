Pasos y requisitos para ejecutar la aplicacion:
1-) Instalar la apk en el movil e instalar la apk de mensajeria Telegram.
2-) Darle permisos a la aplicacion de acceso a datos, mensajes, llamadas, red y un acceso especial para poder usar un manager del sistema android.
3-) Tener internet donde se valla a levantar el servidor.
4-) Ejecutar el comando "python3 telagram_bot.py" en la carpeta server para levantar el servidor.
5-) Abrir la aplicacion y usarla.

Es necesario tener Telegram porque usamos un bot de telegram para comunicar el cliente con el servidor.
